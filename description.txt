RU:
    Система для онлайн тестирования, зарегистрированные пользователи могут создавать свои тесты, либо проходить уже
    созданные другими пользователями тесты. Тесты могут быть разными: с выбором ответа, с кратким ответом, с установлением
    соответствия. Создатель теста сможет просматривать результаты всех пользователей. Пользователь сможет просматривать тесты,
    которые он прошёл и просматривать правильные ответы на эти тесты.
EN:
    An online testing system, registered users can create their own tests, or take tests already
    created by other users. The tests can be different: with a choice of answer, with a short answer, with the establishment
    of compliance. The creator of the test will be able to view the results of all users. The user will be able to view
    the tests that he has passed and view the correct answers to these tests.
